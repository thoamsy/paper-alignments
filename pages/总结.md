# 总结

通过 YSE 的实现, 学习到了如何编写爬虫, 在爬取 Stack Overflow 的过程中, 总是会碰到 429 的问题, 最后学习了一些反爬虫的基本方法, 才做到正常爬取. 通过松弛策略讨论了一个的可行的网页排名算法. 同时通过使用 Koa 接触了 Nodejs 的服务器编程的冰山一角, 使用了 React 和 ECMAScript 最新版本来编写前端代码. 并在 Bulma 这个 CSS 库的帮助下, 省去了很多繁琐的样式编写过程.

在收获的同时, YSE 也存在很多不足. 它的爬虫规模还不够大, 也不够智能. 索引的空间效率也不够高, 需要占很大的空间. 像市面上常见的自动纠错, 自动翻译, 自动联想这些功能它也不具备. 但是作为一个了解搜索引擎原理的项目, 它已经完成了它的目的. 在后续可以查看 Google 的论文来优化算法, 引入机器学习来提升搜索的准确性. 同时, 历史搜索记录, 搜索建议这些基本功能, 也可以加入进来.

技术实在演变的太快, 要保证在自己的领域不拉下的情况下, 还有随时保持对其他技术的敏感度和关注度. 三年之内, 我们从 VR 到 AR, AR 到 AI, AI 再到区块链, 谁能想象五年后的科技会是什么样子呢?
